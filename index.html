    Introduce #

    안녕하세요 🙌 기록하는 주니어 개발자 이미현입니다. 새로운 것을 경험하고, 기록하고, 정리하는 것을 좋아합니다. 2022년 현재 삼성 청년 소프트웨어 아카데미(이하 SSAFY) 7기에 입과하여 소프트웨어 엔지니어로서 더 큰 성장을 위해 열심히 노력하고 있습니다.

    프로세스 자동화와 개발 생산성 향상에 관심이 많습니다. 임베디드 소프트웨어 개발자로 재직하는 동안, 리눅스 및 vim 개발 환경에서 생산성 향상을 위해 Shell Script 와 Makefile를 적극적으로 활용한 경험이 있습니다. 또한, Jira 와 confluence 도입 시 test suite 관리를 담당하여 기존에 문서로 관리하던 시험 데이터를 웹 클라우드 환경에서 관리할 수 있도록 기여했습니다.

    Open Source Project 를 이용한 개발 경험이 있습니다. 임베디드 소프트웨어 개발자로 재직하는 동안 오픈소스 (OpenSSH, TELNET, login, PAM, Syslog-ng) 를 이용한 보안기능 개발을 담당하였습니다.

    다양한 경험을 바탕으로 개발에 대한 이해도가 높습니다. 임베디드 개발 경험으로 얻은 OS 및 프로그래밍 지식을 바탕으로 현재는 SSAFY 에서 웹 프론트엔드, 백엔드에 대한 공부를 하고 있습니다.
